Este repositório foi criado para documentar minha jornada de aprendizado em **React.js**. Aqui guardo anotações, pequenos projetos, exercícios de lógica e experimentos com o ecossistema da biblioteca.

---

## 🚀 Objetivo
O foco principal é dominar desde os conceitos fundamentais (JSX, Props, State) até tópicos avançados como Hooks customizados, Context API e integração com APIs externas.

---

## 📚 Conteúdos Estudados

### 1. Fundamentos
- [x] JSX e Expressões
- [x] Componentização (Functional Components)
- [x] Props e Renderização de Listas (`map` e `filter`)
- [x] Estilização (CSS Modules, Styled Components)

### 2. Hooks (O Coração do React)
- [x] `useState`: Gerenciamento de estado local.
- [x] `useEffect`: Lidando com efeitos colaterais e ciclo de vida.
- [ ] `useContext`: Compartilhamento de dados entre componentes.
- [ ] `useMemo` & `useCallback`: Otimização de performance.

### 3. Ecossistema & Ferramentas
- [x] Vite (Build Tool)
- [ ] React Router (Navegação)
- [ ] Consumo de API (Fetch & Axios)
- [ ] Formulários com React Hook Form
