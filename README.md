# ⚛️ Meus Estudos de React

Bem-vindo ao meu repositório de estudos dedicado ao ecossistema **React.js**. Aqui documento minha jornada de aprendizado, desde os conceitos fundamentais até o desenvolvimento de aplicações completas e escaláveis.



## 🎯 Objetivos do Repositório
Este espaço serve como um diário de bordo técnico para:
* Praticar a sintaxe e as funcionalidades do React.
* Documentar a resolução de desafios e bugs comuns.
* Criar um guia de referência rápida para projetos futuros.

---

## 🚀 Tecnologias e Ferramentas
Durante os estudos, utilizo o seguinte ecossistema:

| Categoria | Tecnologias |
| :--- | :--- |
| **Framework/Lib** | React 18+, Vite, Next.js |
| **Linguagem** | JavaScript (ES6+), TypeScript |
| **Estilização** | CSS Modules, Styled Components, Tailwind CSS |
| **Gerenciamento de Estado** | Context API, Redux Toolkit, Zustand |
| **Consumo de API** | Axios, Fetch API, React Query (TanStack Query) |

## 📚 Tópicos Estudados
Abaixo estão os conceitos que já explorei ou estou explorando atualmente:

### ✅ Fundamentos
* JSX e Virtual DOM
* Componentização e Props
* Rederização de Listas e Condicionais

### 🎣 Hooks (Essenciais e Avançados)
* `useState` e `useEffect`
* `useContext` (Gerenciamento de contexto)
* `useRef` e `useMemo` / `useCallback` (Performance)
* Custom Hooks (Criação de hooks personalizados)

### 🗺️ Roteamento e Navegação
* React Router DOM (Single Page Applications)

### 🧪 Testes e Qualidade
* Vitest / Jest
* React Testing Library

## 🛠️ Como rodar os projetos localmente

1.  Clone o repositório:
    ```bash
    git clone https://github.com/ralphdosreis/react-curso.git
    ```
2.  Entre na pasta do projeto desejado:
    ```bash
    cd nome-do-projeto
    ```
3.  Instale as dependências:
    ```bash
    npm install
    ```
4.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
